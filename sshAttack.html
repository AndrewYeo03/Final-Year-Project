  1. SSH Attack

Student Task:
1) Students are expected to use this pre-prepared script to execute the brute force attack.

2) The script has been configured to run with the preloaded users.txt and passwords.txt files, ensuring a controlled environment for the exercise.

With this setup, students must successfully brute force the SSH credentials to uncover the flag, which is the correct username and password combination. Only by successfully obtaining these credentials through the brute force process will they be able to submit the flag. If students fail to crack the SSH credentials, they will not be able to access or submit the flag, emphasizing the importance of mastering the brute force technique.
